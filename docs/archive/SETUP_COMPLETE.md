   - 如果有图标显示，说明安装成功！

## 🔧 故障排除

### 问题 1: 扩展加载失败

**症状**: 提示 "无法加载扩展"

**解决方法**:
1. 确认选择的是 `extension` 文件夹（不是整个项目）
2. 检查 `manifest.json` 是否存在
3. 查看错误提示，按提示修复

### 问题 2: 图标不显示

**症状**: 扩展加载成功但没有图标

**解决方法**:
1. 确认 3 个图标文件都在 `extension\icons\` 文件夹中
2. 文件名必须是：`icon16.png`, `icon48.png`, `icon128.png`
3. 在扩展页面点击刷新按钮（🔄）

### 问题 3: 无法生成邮箱

**症状**: 点击按钮后提示错误

**解决方法**:
1. 确认后端服务正在运行（应该有一个命令行窗口显示 "临时邮箱服务运行在..."）
2. 如果服务停止了，重新运行 `start.bat`
3. 按 F12 打开浏览器控制台查看详细错误

### 问题 4: 后端服务停止

**症状**: 命令行窗口关闭或服务停止

**解决方法**:
```bash
# 重新启动服务
cd e:\zhuan\CascadeProjects\windsurf-project\windsurf-auto-register\backend
node server.js
```
