## 🚀 如何部署

### 方法1：Vercel自动部署（推荐）
1. 访问 [Vercel Dashboard](https://vercel.com/dashboard)
2. 找到你的项目：`windsurf-backend`
3. 点击 **"Redeploy"** 按钮
4. 等待部署完成

### 方法2：手动上传
1. 打开 Vercel 项目设置
2. 进入 "Deployments" 页面
3. 点击 "Deploy" 按钮
4. 选择 `backend` 文件夹上传

### 方法3：Git推送（需要Git仓库）
```bash
cd backend
git add .
git commit -m "UI 2.0: 全面升级设计系统"
git push origin main
```
