## 🚀 API 更新

### 新增 API

#### 1. 获取服务列表
```http
GET /api/services

响应:
{
  "success": true,
  "services": [
    {
      "name": "NiMail",
      "displayName": "NiMail (推荐)",
      "domain": "@nimail.cn",
      "description": "中文服务，稳定快速",
      "available": true,
      "recommended": true
    },
    ...
  ]
}
```

#### 2. 指定服务生成邮箱
```http
POST /api/generate-email
Content-Type: application/json

{
  "service": "Maildrop"
}

响应:
{
  "success": true,
  "email": "abc123@maildrop.cc",
  "service": "Maildrop",
  "webUrl": "https://maildrop.cc/inbox/abc123",
  "info": "请访问网页查看邮件（完全公开）"
}
```
