## 📦 Step 1: 准备扩展包

### 1.1 检查 manifest.json

确保 `extension/manifest.json` 包含所有必要信息：

```json
{
  "manifest_version": 3,
  "name": "Windsurf 自动注册助手",
  "version": "2.0.0",
  "description": "自动注册 Windsurf IDE，支持临时邮箱、动态码激活",
  "permissions": [
    "storage",
    "activeTab",
    "scripting"
  ],
  "host_permissions": [
    "https://codeium.com/*"
  ],
  "icons": {
    "16": "icons/icon16.png",
    "48": "icons/icon48.png",
    "128": "icons/icon128.png"
  },
  "action": {
    "default_popup": "popup.html",
    "default_icon": {
      "16": "icons/icon16.png",
      "48": "icons/icon48.png",
      "128": "icons/icon128.png"
    }
  },
  "content_scripts": [
    {
      "matches": ["https://codeium.com/*"],
      "js": ["content.js"],
      "run_at": "document_end"
    }
  ]
}
```

### 1.2 准备图标（重要！）

Chrome 商店要求提供多种尺寸的图标：

**扩展图标（必需）：**
- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

**建议尺寸：**
- 图标应该简洁、清晰
- 背景透明（PNG格式）
- 设计要专业

### 1.3 打包扩展

**方法1：直接压缩（推荐）**

1. 进入 `extension` 文件夹
2. 选择以下文件和文件夹：
   ```
   extension/
   ├── manifest.json
   ├── popup.html
   ├── popup.js
   ├── content.js
   ├── floating-panel.html
   ├── background.js
   └── icons/
       ├── icon16.png
       ├── icon48.png
       └── icon128.png
   ```
3. 右键 → **压缩为 ZIP** → 命名为 `windsurf-extension.zip`

**⚠️ 注意：**
- 不要压缩整个 `extension` 文件夹，而是压缩文件夹**内部**的文件
- ZIP 文件打开后应该直接看到 `manifest.json`，而不是文件夹
- 不要包含 `.git`、`node_modules` 等无关文件

## ⏰ Step 6: 等待审核

### 审核时间
- **首次审核**: 通常 1-3 个工作日
- **更新审核**: 通常 1-2 个工作日

### 审核状态
- **待审核**: 正在队列中
- **审核中**: Google 正在检查
- **已发布**: 审核通过，已上线
- **被拒绝**: 审核未通过，需要修改

### 常见拒绝原因
1. **功能不完整或有Bug**
2. **违反 Chrome 商店政策**
3. **权限请求不合理**
4. **描述不准确或误导**
5. **图片质量差或内容不符**
6. **缺少隐私政策**

## 🆘 常见问题

### Q1: 审核被拒绝怎么办？
**A**: 仔细阅读拒绝原因，修改后重新提交。通常问题集中在权限、隐私政策和功能描述上。

### Q2: 审核要多久？
**A**: 首次审核 1-3 天，更新审核 1-2 天。复杂扩展可能需要更长时间。

### Q3: 可以先私有发布吗？
**A**: 可以！选择"不公开"或"私有"选项，审核通过后可以改为公开。

### Q4: 需要提供源代码吗？
**A**: 不是必须，但如果审核团队要求，需要配合提供。

### Q5: 如何应对负面评论？
**A**: 礼貌回复，如果是Bug及时修复。不要与用户争论。

### Q6: 更新会自动推送吗？
**A**: 是的，审核通过后会自动推送给已安装用户（可能需要重启浏览器）。
