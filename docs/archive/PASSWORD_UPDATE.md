## 🚀 使用方法

### 步骤 1: 重新加载扩展

```
1. chrome://extensions/
2. 找到 Windsurf Auto Register
3. 点击刷新图标 🔄
```

### 步骤 2: 开始注册

```
1. 打开 https://windsurf.com/account/register
2. 点击扩展图标
3. 点击 "开始自动注册"
```

### 步骤 3: 查看密码

```
扩展弹窗会显示:
- 临时邮箱
- 生成的密码 ⭐ 新功能

两个都可以复制！
```

## 🔧 故障排查

### 问题: 密码没有显示

**解决**:
```
1. 确认已重新加载扩展
2. 刷新注册页面
3. 重新点击 "开始自动注册"
```

### 问题: 密码没有自动填写

**解决**:
```
1. 按 F12 查看控制台
2. 查看是否有 "密码输入框已填写" 日志
3. 如果没有，手动复制密码填写
```
