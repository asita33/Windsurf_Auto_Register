# 🚀 Chrome 商店发布快速清单

## ⚡ 5分钟发布准备

### 1️⃣ 准备图标（必需）

在 `extension/icons/` 文件夹中准备：
- [ ] `icon16.png` - 16x16 像素
- [ ] `icon48.png` - 48x48 像素  
- [ ] `icon128.png` - 128x128 像素

**💡 快速制作工具：**
- Canva: https://www.canva.com/
- Figma: https://www.figma.com/
- 或使用在线图标生成器

### 2️⃣ 打包扩展

```bash
# 进入 extension 文件夹
cd extension

# 压缩所有文件为 ZIP（不是压缩文件夹本身！）
# Windows: 选择所有文件 → 右键 → 发送到 → 压缩文件夹
# Mac: 选择所有文件 → 右键 → 压缩
```

**⚠️ 重要：ZIP 打开后应直接看到 `manifest.json`**

### 3️⃣ 准备截图（至少1张）

**尺寸要求：** 1280x800 或 640x400 像素

**建议截图：**
- [ ] 弹出窗口界面
- [ ] 功能演示
- [ ] 成功状态

### 4️⃣ 注册并上传

1. 访问: https://chrome.google.com/webstore/devconsole
2. 登录 Google 账号
3. 支付 $5 注册费（一次性）
4. 点击"添加新项"
5. 上传 ZIP 文件
6. 填写商店信息
7. 提交审核
